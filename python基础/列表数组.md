## 列表相关操作

- #### 列表去重

  ```python
  import reduce
  
  
  def list_distinct():
      """
      一般列表去重，且不改变原来顺序
      """
      list_1 = [1, 4, 1, 9, 7, 4, 'aa', 'bb', 'aa']
      list_2 = sorted(set(list_1), key=list_1.index)
      return list_2		# [1, 4, 9, 7, 'aa', 'bb']
  
  
  def list_dict_distinct():
      """
      列表字典去重，不改变原本顺序,字典元素完全相同时认为重复
      其原理就是列表生成是看新的元素是否存在于列表中
      reduce() 函数会对参数序列中元素进行累积。
      函数将一个数据集合（链表，元组等）中的所有数据进行如下操作：
      用传给 reduce 中的函数 function（有两个参数）先对集合中的第 1、2 个元素进行操作，
      得到的结果再与第三个数据用 function 函数运算，最后得到一个结果。
      """
      list_1 = [{'id': 1, 'name': '小李'}, {'id': 2, 'name': '小明'},
                {'id': 3, 'name': '小王'}, {'id': 1, 'name': '小李'}]
      list_2 = reduce(lambda x, y: x if y in x else x + [y], [[], ] + list_1)
      return list_2
  
  
  def dedupe(items, key=None):
      """
      列表字典去重，根据字典某个或多个键值对去重
      :param items: 字典列表
      :param key: 匿名函数,获取判定是否重复的值
      """
      seen = set()
      for item in items:
          val = item if key is None else key(item)
          if val not in seen:
              yield item
              seen.add(val)
  
  
  if __name__ == '__main__':
      list_1 = [{'id': 1, 'name': '小李'}, {'id': 2, 'name': '小王'},
                {'id': 3, 'name': '小王'}, {'id': 1, 'name': '小李'}]
      r1 = list(dedupe(list_1, key=lambda x: x['name']))
      r2 = list(dedupe(list_1, key=lambda x: (x['id'], x['name'])))
      print(r1)   # [{'id': 1, 'name': '小李'}, {'id': 2, 'name': '小王'}]
      print(r2)   # [{'id': 1, 'name': '小李'}, {'id': 2, 'name': '小王'}, {'id': 3, 'name': '小王'}]
  ```

- #### 列表字典更新

  ```python
  def list_dict_update():
      """
      列表字典中，对每个字典进行更新
      """
      list_1 = [{'id': 1, 'name': '小李'}, {'id': 2, 'name': '小明'}]
      [x.update(name='小哈哈') for x in list_1]
      return list_1	# [{'id': 1, 'name': '小哈哈'}, {'id': 2, 'name': '小哈哈'}]
  ```
